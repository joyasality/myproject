Pertama-tama Alhamdullilah, Allah memberikan kemudahan untuk saya menyelesaikan aplikasi ini.
Untuk menggunakan aplikasi ini Anda memerlukan software untuk menjadikan komputer Anda server.
Dalam aplikasi ini, saya menggunakan XAMPP ver 1.6.6a. XAMPP adalah
program yang berisi paket aplikasi PHP-Apache dan Mysql yang sudah
terintegrasi dimana instalasi untuk ke-3 aplikasi tersebut hanya dilakukan sekali saja

Langkah-langkah yang harus Anda lakukan adalah :
1.  Ekstrak file XAMPP sehingga menjadi xampp-win32-1.6.6a-installer.exe
2.  Instal XAMPP dengan cara mendouble klik XAMPP dan pilih lokasi instalasi di d:\
3.  Maka akan ada direktori baru di d:\ yaitu d:\xampp
4.  Setelah proses instalasi selesai lihat di direktori instalasi tersebut (xampp) dan jalankan file xampp-start.exe
5.  Akan muncul jendela ms-dos dan memberitahukan bahwa program is starting now (minimize jendela tersebut, jangan di tutup)
6.  Direktori kerja berada di dalam folder instalasi -> htdocs -> xampp  atau d:\xampp\htdocs\
7.  Buatlah folder baru sesuai dengan nama anda, misal : cenat-cenut, maka path direktori kerja adalah d:\xampp\htdocs\cenat-cenut
8.  Ekstrak file cenat-cenut simpan di direktori d:\xampp\htdocs\cenat-cenut
9.  Untuk menggunakan aplikasi ini, Anda harus membuat database terlebih dahulu. caranya adalah dengan menggunakan command prompt atau phpmyadmin
10. Gunakan PhpMyAdmin agar lebih mudah. Buka Mozilla, Internet Explorer atau browser lainnya. Kemudian ketikkan pada addressbar localhost/phpmyadmin, enter.
11. Ketikkan nama database pada create new database, kemudian tekan create.
12. Klik tab Import, kemudian Browse..pilih file pakarcinta.sql kemudian go.
13. Kemudian Setting konfigurasi untuk koneksi ke database. pilih pakarcinta\include\inc.koneksidb.php. Kemudian edit nama database dengan nama database yang
	telah Anda buat,gunakan password bila perlu.
14. Kemudian Ketikkan pada addressbar mozilla localhost/nama_folder_Anda contoh localhost\cenat-cenut
15. Aplikasi telah siap digunakan.
